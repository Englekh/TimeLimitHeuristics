## About The Project

This is a repository for code used in "Shedding Light on Complex Bitcoin Mixer Transactions: 67-Fold
Reduction in Unclassified Cases". This repository has code for bitcoin blockchain parsing and transaction classification. 

## Files

Heuristics.ipynb - Main notebook containing all heuristics implementation and testing pipeline
Transactions.jsonl - subset of transactions as example
Solver.out - transactions classfificator, taken from previous project [link](https://github.com/Englekh/shared-send-classification)
